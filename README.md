# Browser Game Playset

一个多原型网页游戏合集：`/` 是游戏选择大厅，`脑花守卫` 是路径塔防肉鸽，`林间小屋` 是 3D 生存建造原型，`深岩工坊` 是体素挖矿建造原型，`星环防线` 是 3D 轨道防御原型。

建议 GitHub 仓库名：`browser-game-playset`。

## 入口

- 打开 `/` 先进入游戏大厅，不再默认启动某一个游戏。
- 打开 `/?game=tower` 进入 `脑花守卫`。
- 打开 `/?game=cabin` 进入 `林间小屋`。
- 打开 `/?game=voxel` 进入 `深岩工坊`。
- 打开 `/?game=orbit` 进入 `星环防线`。

## 脑花守卫

- 僵尸沿固定蛇形道路前进，终点是脑子，漏怪会扣脑子血。
- 开局会自动铺一套守门阵，玩家可以直接开波，也可以点击底部植物卡补位。
- 底部“巡航”会一键托管补种、拿卡、自动开波和节奏提速，适合不想每波重复操作的局。
- 默认启用“指挥官节奏”：开局后会自动整备和开波，战利品会自动拿推荐卡；玩家仍可关掉自动开波改成手动节奏。
- 默认启用“战役管家”：它会把补种、强化、预案、契约、节奏和波中救场压成每波 1 个主决策预算，并在 HUD 里说明本波为什么这么打、热点在哪里、节省了多少重复点击。
- 默认启用“低操作导演”：每波玩家负担固定压到 1 个主决策，HUD 会显示玩家决策数、管家代办步数和目标压力带，避免巡航几波后又退回一堆按钮。
- 画布底部会显示“焦点指令”：它会根据当前阶段一键执行开局、巡航接管、开波、波中救场、补强收尾或自动拿推荐卡，并计入省点击，不需要玩家在多个面板里找下一步。
- 默认启用“焦点巡航”：进入巡航后会按战局心跳自动执行焦点指令，低压波自动观察整备，高压波自动救场，战利品阶段自动拿卡，让玩家不必每波盯着按钮。
- 默认启用“免手波次”：巡航或自动模式下，拿卡、整备、开波、救场会归入同一条免手账本，HUD 显示免手次数，debug 会给出当前决策负担，目标是让连续波次保持 0 手动负担。
- 作战计划默认开启：每波前会自动评估下波压力、补种关键空位、强化核心植物，必要时预购护盾，并显示节省了多少重复点击。
- 战术队列默认开启：每波把补位、强化、护盾、导演救场、自动选卡和开波收束成一条计划；HUD 会显示队列动作、平衡判定和下一步建议，避免玩家每波重复点一串按钮。
- 作战预案默认开启：每波会在“稳线 / 滚经济 / 强杀”之间自动选择一个意图，右侧三张预案卡也能直接点击执行；稳线会买护盾或冰网救场，滚经济会留足防守预算后再补向日葵、增产和悬赏，强杀会升级主力并斩前锋，把每波操作从一堆零散按钮压成一个真实决策。
- 构筑路线会自动识别当前更像藤蔓控场、玉米爆破、日照经济、豌豆火线还是混编成长；战利品推荐会按路线加权，HUD 简报会直接提示下一步该拿什么。
- 波次契约会给每一波自动派发一个小目标，例如完美守门、压住终点、阳光悬赏或猎杀前锋；完成会给阳光、护盾、悬赏或攻速奖励，失败也会在复盘里讲清原因。
- 作战手牌默认开启：右上角会把本波路线热点、补位方向和推荐预案压成 3 张短牌；三张牌分别执行压热点、补位/强化、推荐预案，不再是三个按钮共用一套泛化逻辑。
- 波次蓝图会按训练、雾跑先锋、铁桶护卫、群尸潮、终点围压和 Boss 围城切换尸潮主题；路径上会显示本波蓝图和应对建议，刷怪 mix 也会跟着主题变。
- 路线压力会按入口、上弯、中庭、终点计算热点，在路径上显示压力数值和需要的应对，例如补寒冰、补玉米、补护盾或守拐点。
- 节奏调度会根据战力/威胁比和波中压力自动快进或刹车；优势局少等，吃紧局降速给导演救场。
- 平衡导演会把战力/威胁比夹在每波目标压力带附近：溢出时会按缺口补怪量、提升精英压力，必要时锁住过量阳光；偏弱时补护盾和阳光。平衡账本会记录夹紧前后、公平债、怜悯库、精英倍率和最近调节，方便看出为什么这一波变难或变稳。
- 战斗复盘会记录本波伤害、击杀、阳光收益、导演救场、漏防和波次评价；右侧会显示 MVP 植物和贡献榜，植物角标也会显示等级或击杀数。
- 自动园丁默认开启：每次开波前会按阳光预算自动补关键空位，减少每波重复种植。
- 自动策略默认开启：战利品阶段会短暂停留展示推荐卡，然后自动选择；玩家也可以关掉后手动选流派。
- 战局导演会估算下一波压力，阵容明显偏弱时给少量护盾和阳光兜底，降低因为一次误点直接崩盘的概率。
- 战局导演还会下“现场指令”：每波自动空投植物、火力超频、临时冰网、应急护盾或补给包；这些指令会改变战场状态并计入省点击。
- 自适应强度会根据上波评价校准后续尸潮：连续优势会轻微加压，漏防或救场偏多会降压，避免固定数值曲线忽然失衡。
- 顶部信息栏会显示下波预警、战力/威胁评分和导演状态，玩家能看懂为什么该补位或为什么可以巡航。
- 路径上会显示下一波的压力标记，例如硬、快、群、甲、王；每波结束会给出波次评价并补给阳光。
- 每波会触发战场事件：补给雨提升阳光产出，藤蔓疯长强化近路植物，薄雾突袭让前锋加速，夜袭压迫会增加怪量和混编压力；事件会在路径上出现可视标记。
- 点击底部植物卡，再点击空土格种植；`Esc` 或点击空白处取消当前选择。
- 点击已经种下的植物可以查看它的范围、等级和最近强化。
- 植物卡包括：豌豆、向日葵、寒冰、玉米、地刺。
- 向日葵产阳光，击杀僵尸也会掉阳光，阳光继续买植物。
- 每波结束后，全体已种植物会自动成长 1 级，不需要逐棵重复点击。
- 每波只从 3 张策略卡里选 1 张：一张偏战术低点击操作，一张偏流派成长，一张偏通用补强。
- 策略卡可以空投植物、复制强力植物、开启自动巡航、加护盾、加悬赏阳光或改变全局火力。
- 底部右侧可以切换自动开波和速度，减少重复操作。
- 守住 12 波，最后一波有 Boss。

## 林间小屋原型

- 打开 `?game=cabin` 进入 3D 林间求生原型。
- 开场会以暗色森林遮罩进入，小屋、炉火和树林先成为第一眼焦点；资源标签、蓝图地垫和事件信标会先隐藏，点击开始后再恢复完整 HUD 与任务引导。
- 开场会显示第一夜剧本，把拾荒、制斧、点火和侦察压成一组短目标，避免一进入林地不知道先干什么。
- 小屋有暖窗、门廊灯和缓慢飘起的烟囱烟，开场更像一个已经有人住下的林间据点。
- 开始后会显示中心准星、蓄力环和就近提示，跟随当前材料、最近目标和是否可建造变化，3D 场景不再只靠边栏读信息。
- 准星会锁定玩家附近的可采集资源，显示 `E` 可以拾取或采集的对象和距离；按 `E` 会优先执行就近交互，没有目标时才退回吃浆果。
- 手持条会记录最近一次就近交互，例如 `E 采集石堆 0.5m`，资源、准星和手持反馈会一起更新，采集不再像静默加数值。
- 镜头右下会显示近景手持工具或当前建材，靠近资源时会自动切成石斧、石镐、木弓或徒手动作，采集瞬间会和 3D 挥动轨迹一起动。
- 最近资源会在 3D 场景里出现双层锁定圈和 `E` 标签，采集时会飘出收益文字；狼踪或夜间狼压会在地面铺出脚印，寒冷/狼压/饥饿会压暗画面边缘。
- 采集会在角色手边打出短暂挥动轨迹，小屋周围会有营地安全圈；安全圈会根据寒冷、狼压和屋况切换色调，让风险不用只靠面板读数判断。
- 开始后会显示行动焦点条，把推荐营地指令、当前风险和目标距离合成一个可点击的下一步；点击后会执行真实营地指令，不只是提示文案。
- 林间小屋会写入浏览器本地快照，保存资源、状态、已建方块、事件、日程和营地位置；再次打开时会优先恢复快照里的营地进度。
- 场景里有小屋、树、岩石、煤矿、铁矿、浆果丛、拾荒点、补给箱、野鹿、野猪、游荡者、夜晚出没的狼，以及雨天和寒潮天气。
- 点击树、岩石、浆果丛或拾荒点采集资源；点击地面消耗资源建造木墙、石基、树篱、窗格、营灯、木箱、熔炉和床铺。
- 点击“执行任务”会自动推进当前任务：拾荒、砍树、制作、点篝火、建造、扎营入夜、制弓、打猎、烤肉、制甲、做石镐、采矿、炼铁、整理木箱、铺床、侦察地图、扛过寒潮、睡过长夜、建设施、开启管家托管和派远征会串成一条低操作流程。
- 右上角小地图会显示玩家、小屋、当前目标、林地事件和已侦察资源；点击小地图会展开地图层并补齐煤矿、铁矿、浆果和夜狼径。
- 地面会出现 3D 金色路径点和目标圈，自动指向当前任务或紧急风险位置；底部手持条会显示目标距离。
- 夜晚和雨雾里会有环绕小屋的萤火微光，配合篝火、天气和事件信标强化林地氛围。
- 靠近地上的木棍、石子和补给箱会自动拾取，不需要对着小物件反复点。
- 收集木棍和石头后可以制作石斧，石斧会提高砍树收益。
- 制作木弓后狩猎收益更高，制作皮甲后防护值会明显提高，制作石镐后才能稳定采铁矿。
- 点起篝火后营地会出现火光，并降低夜晚狼群压力。
- 生肉可以在篝火旁烤熟，熔炉会把铁矿和煤炭炼成铁锭。
- 左侧 HUD 会显示生命、饥饿、寒冷、狼压、天气、营地评分、地图情报、资源和任务链。
- 完成侦察后，煤矿、铁矿、浆果和夜狼路径会在 3D 场景里出现地图标记。
- 床铺可以在夜晚跳到清晨，恢复生命并压低寒冷和狼压；寒潮会考验篝火、营灯、围墙、熔炉和床铺组成的保温能力。
- 雨桶、陷阱线和哨塔会作为营地设施出现在 3D 场景里；它们分别改善补给、肉源和夜狼预警。
- 开启“托管”后，营地管家会按节奏自动烤肉、吃浆果、收陷阱、蓄雨桶、压狼压和维持安全评分，减少后期重复操作。
- 点击“自走营地”会持续推进任务链：缺料自动采集、够料自动建造、需要时自动点工具和设施，适合少点击跑完整个生存循环。
- 点击“蓝图铺设”会按门廊、照明、树篱、工坊、床位和围护阶段批量建造；地面会显示半透明蓝图占位，营地不是随机堆块。
- 点击“背包”会打开背包 / 制作台面板；口粮包、营地套件、巡径包、修补包和保温包会消耗真实资源，转成补给、压狼压、抗寒或回血。
- 林地事件会随任务推进触发，例如旧路补给、雨后蘑菇、狼踪逼近和寒锋压林；事件会实际改变补给、狼压、寒冷或天气，并在 3D 场景里出现信标。
- HUD 的营地计划会把当前林地事件、风险优先级和下一步目标串起来，减少不知道该点什么的问题。
- 小屋状态会用屋况面板显示干燥、余温、噪音和士气，直接标出当前短板；日程总管会优先修屋顶、补炉火、静音巡线或整理床铺，雨天、寒潮、狼踪、背包和管家维护也都会改变屋况，并影响寒冷、狼压和安全评分。
- 右侧“营地指令”会把巡林采集、蓝图加固、守夜调度和日程总管压成 4 张指令卡；推荐卡会跟随饥饿、寒冷、狼压、地图和防线风险切换，点击后会真实采集、铺蓝图、建项目、侦察、睡觉、触发管家维护，或一次串起吃饭/保温/守夜/扩建的每日计划。
- 快捷栏会显示数字角标，底部手持条、角色手持物和地块预览会同步当前材料、下一目标和是否可建造；树木、石堆、矿点、浆果、动物和补给会在 3D 场景里显示小标签。
- 点击“远征”会按林线、矿脊、猎径顺序派队带回资源；补给、护甲、哨塔和陷阱线会降低远征代价。
- 吃浆果可以恢复饥饿，营灯和树篱能降低夜晚压力。
- 右键删除已放置方块，数字键 `1-8` 切换材料。
- 拖拽旋转视角，滚轮缩放，`WASD` 移动幸存者和镜头目标。

## 深岩工坊

- 打开 `?game=voxel` 进入 3D 体素挖矿建造原型。
- 场景里有地底核心、岩层、矿脉、晶体、裂隙和矿袭敌人。
- 左键挖方块，右键在地面建造当前选中的岩块、桥板或晶炮。
- 点击“蓝图托管”会连续推进当前目标：挖岩、找矿、采晶、精炼、搭桥、造炮、启动矿钻、立能量塔、装载研究和抵御多轮矿袭。
- 工坊会显示可视化建造蓝图：桥板、防线、矿钻和能量塔都有半透明规划地垫和阶段标签，未挖空的位置会提示先挖。
- 设施建成后，核心会向晶炮、矿钻、桥板和能量塔拉出供能/物流线；电网评分会参与工坊评分，不再只是“造了几个建筑”的数值。
- 右侧“工序指挥”会把清障采掘、精炼补料和蓝图施工压成 3 张工序卡；面板会显示当前挖掘焦点、蓝图阻挡点和材料短缺，一键工序会真实切换采掘、精炼或施工。
- 地面会显示挖掘提示环和阶段标签，例如先清掉蓝图阻挡点、优先挖富矿或晶簇，玩家不用猜下一镐该敲哪里。
- 点击“矿脉扫描”会标出碎岩空腔、富矿口袋、晶簇花园或躁动裂隙；后续匹配采掘会实际追加碎岩、矿石、合金、晶体、能量或蓝图数据。
- 点击“执行工序”仍可单步推进，适合想半自动控制节奏时使用。
- 精炼会把矿石转成合金，合金用于桥板和晶炮，能量主要留给防御建筑。
- 矿钻会自动产出矿石、碎岩和蓝图数据，能量塔会提高防御评分并补充能量。
- 研究协议包括脉冲扫描、深钻齿轮、晶核稳压、核心护甲和轨桥工法，会实际影响产出和矿袭防守。
- 矿袭会按碎钻蜂群、护盾矿兽和晶尘风暴变招；敌人数量、威胁、漏伤和奖励蓝图都会跟事件类型变化。
- HUD 会显示工坊评分 / 威胁评分、当前矿袭事件、研究等级、扫描矿脉和蓝图托管步数。
- 拖拽旋转视角，滚轮缩放，`WASD` 移动观察目标，数字键 `1-5` 切换建造物。

## 星环防线

- 打开 `?game=orbit` 进入 3D 轨道防御原型。
- 场景里有行星核心、三层轨道、激光卫星、护盾卫星、回收卫星和入侵无人机。
- 卫星会自动绕行，入侵波次由系统结算火力、护盾和漏伤。
- 点击“执行战术”会自动推进当前目标：部署火力网、补护盾、建回收、缺料时自动迎击、升级主炮和连续迎击。
- 右侧“轨道指挥”会把下一波预测压成外环火线、中环拦截、近地护盾和回收补给热点；推荐的 3 张战术卡可以一键切编队、补节点、维修护盾、滚经济或直接迎击。
- 点击“战术编队”会在拦截扇面、护盾环阵、回收拖网和过载脉冲之间切换；编队会实际改变火力、护盾、回收收益和耗能。
- 每波会触发轨道事件，例如太阳风顺流、陨石雨、离子风暴和掠夺者突袭；事件会改变威胁、火力、护盾、残骸和能量奖励，并在危险轨道上显示标记。
- 下一波压力会以 3D 弧线标在对应轨道上，战斗日志会记录威胁、火力、护盾、漏伤、事件、编队和当前热点，玩家不用猜为什么要补火力或蓄盾。
- 每次迎击后会自动装载一项星环协议，例如磁轨透镜、残骸税、蔚蓝壁垒、电容阵列、维修蜂群和星链编队，协议会实际影响火力、回收、护盾、补能和维修。
- 战斗会显示激光束、护盾冲击环、外圈危险轨道和残骸带，不再只是静态模型。
- 点击“迎击”可以手动开始下一波，数字键 `1-3` 切换要部署的卫星。
- 拖拽旋转视角，滚轮缩放，`WASD` 移动观察目标。
